# Documentação Técnica - Tempero da Vó

## Requisitos Funcionais (RF)
- RF-01: Exibir cardápio com preços.
- RF-02: Adicionar itens ao carrinho.
- RF-03: Calcular pontos de fidelidade (1 ponto por R$ 10).

## Requisitos Não Funcionais (RNF)
- RNF-01: Interface com alto contraste para acessibilidade.
- RNF-02: Backend em Node.js com TypeScript.
- RNF-03: Frontend em Angular.

## Regras de Negócio (RN)
- RN-01: Pedido mínimo de R$ 20,00.
