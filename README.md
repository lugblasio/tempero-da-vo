# 👵 Tempero da Vó - Sistema de Delivery

Projeto acadêmico de delivery focado em acessibilidade e fidelização de clientes.

## 🛠️ Tecnologias
- **Back-end:** Node.js + TypeScript
- **Front-end:** Angular + TypeScript
- **Documentação:** Requisitos Funcionais, Não Funcionais e MER/DER.

## 📍 Cenário
Localizado no bairro do Ipiranga, o restaurante busca independência de plataformas de delivery, oferecendo uma interface simplificada para clientes de 35 a 60 anos.
