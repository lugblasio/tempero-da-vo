# 👵 App Delivery - Tempero da Vó

Projeto de sistema de delivery focado em acessibilidade para pequenos restaurantes.

## 📝 Cenário
O restaurante **Tempero da Vó** (Ipiranga, SP) busca independência das grandes plataformas de delivery. O público-alvo tem entre 35 e 60 anos, exigindo uma interface simples e intuitiva.

## 🚀 Tecnologias Utilizadas
- **Front-end:** Angular (TypeScript)
- **Back-end:** Node.js (TypeScript)
- **Banco de Dados:** PostgreSQL (ou SQLite para desenvolvimento)
- **Documentação:** Engenharia de Software (Requisitos, MER/DER)

## ♿ Foco em Acessibilidade (WCAG)
- **Perceptível:** Cores de alto contraste e fontes grandes.
- **Operável:** Navegação simplificada com poucos cliques.
- **Compreensível:** Linguagem direta e botões com ícones claros.

## 🛠️ Requisitos do Sistema
### Requisitos Funcionais (RF)
1. Exibir cardápio com fotos caseiras.
2. Carrinho de compras com resumo claro.
3. Programa de fidelidade acumulativo.

### Requisitos Não Funcionais (RNF)
1. Tempo de resposta inferior a 2 segundos.
2. Interface responsiva para celulares antigos.
